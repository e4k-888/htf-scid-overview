## Overview

HTF SCID is designed around higher-timeframe contextual analysis applied to intraday futures trading, primarily focusing on the E-mini S&P 500 (ES) and Nasdaq-100 (NQ).

The model emphasizes structured market context rather than indicator-based signals or mechanical rule sets.
